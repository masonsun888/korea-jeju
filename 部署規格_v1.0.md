## 📋 執行步驟

### 步驟 1：建立 GitHub Repo

```bash
# 確認在工作目錄
cd ~/projects  # 或你習慣的位置

# 建立資料夾
mkdir jeju
cd jeju

# 初始化 git
git init
git branch -M main
```

### 步驟 2：放入 HTML 檔案

把 `jeju-dashboard-v4.html` 複製到資料夾、並**重新命名為 `index.html`**：

```bash
# 假設原檔在下載資料夾
cp ~/Downloads/jeju-dashboard-v4.html ./index.html

# 驗證
ls -la
# 應該看到 index.html
```

### 步驟 3：建立 .gitignore（可選）

```bash
cat > .gitignore << 'EOF'
.DS_Store
*.log
node_modules/
.env
EOF
```

### 步驟 4：建立 README.md

```bash
cat > README.md << 'EOF'
# 〈染上濟州色〉8 人家族旅行 Dashboard

2026/6/24 → 7/01 · 3 對夫妻 + 2 個 4 歲小孩

## 功能
- 📖 行程介紹（每日重點+連結）
- 📅 詳細時間軸
- 🎯 家庭投票（阿Q / Mason / 阿賢）
- 🚗 交通安排
- 🍽️ 訂位清單
- 💆 醫美選項
- 🎰 賭場夜
- 🛍️ 購物指南
- 📞 緊急電話
- ⚙️ Checklist + 記帳

## 預覽
https://masonsun888.github.io/jeju/
EOF
```

### 步驟 5：Commit + Push 到 GitHub

```bash
# 加入所有檔案
git add .
git commit -m "v1.0 initial deploy: 〈染上濟州色〉Dashboard"

# 在 GitHub 上建立新 repo（如果還沒建立）
# 1. 去 https://github.com/new
# 2. Repository name: jeju
# 3. Public（一定要 Public 才能用 Pages）
# 4. 不要勾任何 README / .gitignore / license（避免衝突）
# 5. Create repository

# 連結遠端 repo
git remote add origin https://github.com/masonsun888/jeju.git

# 推送
git push -u origin main
```

### 步驟 6：開啟 GitHub Pages

**用網頁操作（推薦）**：
1. 進入 `https://github.com/masonsun888/jeju`
2. 點上方 **Settings**
3. 左側選單找 **Pages**
4. **Source** 區塊：選 `Deploy from a branch`
5. **Branch** 區塊：
   - Branch: `main`
   - Folder: `/ (root)`
6. 點 **Save**
7. 等 1-2 分鐘、頁面會顯示 ✅ Your site is live at `https://masonsun888.github.io/jeju/`

## 🚨 常見問題

### Q: Push 時遇到認證錯誤？
A: 用 Personal Access Token：
1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. Generate new token → 勾 `repo` 權限 → Generate
3. 複製 token、push 時當作密碼貼上

或設定 SSH key：
```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
cat ~/.ssh/id_ed25519.pub
# 複製內容貼到 GitHub → Settings → SSH and GPG keys
```

### Q: GitHub Pages 顯示 404？
A: 
1. 確認檔名是 `index.html`（不是 `jeju-dashboard-v4.html`）
2. 確認 repo 是 Public
3. 確認 Pages 設定的 branch 和 folder 正確
4. 多等幾分鐘（首次部署有時要 5-10 分鐘）

### Q: 推上去之後網址打不開？
A: 等！第一次部署需要 1-5 分鐘 propagate。可以去 Actions 分頁看部署進度。

### Q: 想自訂網址（例如 jeju.mason.com）？
A:
1. Settings → Pages → Custom domain 輸入 `jeju.mason.com`
2. 你的 DNS 加一筆 CNAME：`jeju` → `masonsun888.github.io`
3. 等 DNS 生效（10 分鐘到 1 小時）
